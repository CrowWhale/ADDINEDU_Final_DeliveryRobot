# Sequence
1. 자율주행 part
> * 배송로봇 자율주행 구현(ROS, Mapping, Navigation)
2. Manipulator
> * manipulator 구동 알고리즘(ROS)

3. Station part

> * 객체 text 데이터 인식(Computer vision, Image processing)
